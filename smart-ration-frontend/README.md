# Smart Ration Frontend

React frontend for the Smart Ration Distribution System.

## Tech Stack

- React
- React Router
- Axios
- Bootstrap
- Vite

## Pages

- Login
- Register
- Dashboard
- Ration Products List
- Order Page

## Run

```powershell
cd c:\Users\Administrator\OneDrive\Documents\Smart-ration-System\smart-ration-frontend
npm install
npm run dev
```

Frontend URL:

```text
http://localhost:5173
```

Backend API base URL:

```text
http://localhost:8080/api
```

## Note

The frontend fetches ration products from the backend inventory API and converts `rice`, `wheat`, and `sugar` stock into product cards.
